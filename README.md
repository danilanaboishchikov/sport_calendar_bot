# Telegram Fitness Bot

Асинхронный Telegram бот для трекинга тренировок, воды и мотивации.

## Структура проекта

```
telegram_bot/
├── main.py                 # Точка входа
├── config.py               # Конфигурация и настройки
├── requirements.txt        # Зависимости Python
├── README.md              # Документация
└── app/
    ├── __init__.py
    ├── ai.py              # Работа с AI (DeepSeek)
    ├── callback.py        # Обработчики callback-запросов
    ├── main_handler.py    # Основные обработчики сообщений
    ├── time_handler.py    # Система напоминаний
    ├── database/
    │   ├── __init__.py
    │   ├── db.py          # Асинхронная работа с SQLite
    │   └── excel.py       # Генерация Excel отчётов
    ├── payment/
    │   ├── __init__.py
    │   └── payment.py     # Работа с YooMoney
    └── util/
        ├── __init__.py
        ├── keyboards.py   # Генерация клавиатур
        ├── messages.py    # Генерация сообщений
        └── utils.py       # Вспомогательные функции
```

## Установка

1. Клонируйте репозиторий или распакуйте архив

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка

Создайте файл `.env` или установите переменные окружения:

```bash
# Обязательные
BOT_TOKEN=your_telegram_bot_token
AI_KEY=your_ai_api_key
ADMIN_ID=your_telegram_id

# Для платежей YooMoney
YOOMONEY_TOKEN=your_yoomoney_token
YOOMONEY_RECEIVER=your_yoomoney_wallet

# Опционально
ACCESS_PRICE=100  # Цена доступа в рублях
```

Или отредактируйте файл `config.py`, заменив заглушки на реальные значения.

## Запуск

```bash
python main.py
```

## Функционал

### Для пользователей:
- 🏃 **Трекинг тренировок** - добавление и учёт тренировок и упражнений
- 📝 **Дневной отчёт** - ежедневное заполнение отчёта о тренировках
- 🫗 **Трекер воды** - учёт выпитой воды с целями
- 🌅 **Утренняя мотивация** - AI-генерируемая мотивация каждое утро
- ⏰ **Напоминания** - настраиваемые напоминания о воде и активности
- 📊 **Excel отчёты** - детальные отчёты с графиками

### Для администратора:
- Одобрение/отклонение запросов на доступ
- Настройка цен и параметров доступа

## База данных

Бот автоматически создаёт SQLite базу данных `bot_database.db` при первом запуске.

### Таблицы:
- `users` - пользователи и их настройки
- `sport` - спортивные активности
- `reports` - дневные отчёты
- `water` - записи о воде

## Технологии

- **Python 3.9+**
- **aiosqlite** - асинхронная работа с SQLite
- **httpx** - асинхронные HTTP запросы
- **pyTelegramBotAPI** - работа с Telegram Bot API
- **xlsxwriter** - генерация Excel файлов
- **qrcode** - генерация QR кодов
